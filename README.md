# Horto Mio Plus - KIT
## Green Lab - Future Education Modena 2022

## Hardware utilizzato
- Arduino MKR WiFi 1010

## Librerie
- MKRENV.h
- Wire.h
- WIFININA.h
- ArduinoJson
- Dallas Temperature

## Sensori
- MKR ENV SHIELD
- Dallas DS18B20 (Temperatura acqua)
- Seeed Studio Grove TDS Sensor
- Seeed Studio Grove Soil Capacitive Sensor
